API reference

Public API of CallSimulator. Returns MIAIRR-style calls and the reference phased genotype.

CallSimulator.ChromosomeWeights — Type

Weights for one chromosome: (gene_index, weight) pairs for expression sampling.

CallSimulator.DonorGenotype — Type
DonorGenotype

Diploid genotype for one donor: two chromosomes, with genes stored per locus (genes_v, genes_d, genes_j). Use genes(gt, V) etc. and zygosity_counts(gt, L). Display in REPL or with show(io, MIME("text/plain"), gt) to see genes and zygosities.

CallSimulator.EmpiricalParams — Type
EmpiricalParams

Tunable parameters (no gene pools): usage skew (lognormal sigma, allele imbalance), anchor J fraction range (min, max) for sampling, and per-locus hemizygosity probabilities (V/D/J). Gene pools are passed separately via GenePools (e.g. Simulator(config, GenePools(), ki_donor_preset())).

CallSimulator.ExpressionProfile — Type
ExpressionProfile

Holds (gene_index, weight) per locus per chromosome. Gene index is within that locus. Access via weights(ep, L, chr); sample via (ep)(rng, gt, L, chr).

CallSimulator.ExpressionProfile — Method

Functors: (ep::ExpressionProfile)(rng, gt, L, chr) -> gene index within genes(gt, L).

CallSimulator.GenePools — Type
GenePools(; v, d, j)

V/D/J gene name vectors for genotype building. Omit arguments to use default gene pools. Use with Simulator(config, GenePools(), ki_donor_preset()) so you can tune params without re-specifying genes.

CallSimulator.NoiseConfig — Type

Call-corruption rates (allele swap, D dropout) per locus. Pass to NoiseModel(cfg).

CallSimulator.NoiseModel — Type

Callable: (rng, gt, L, chr, geneentry, trueallele) -> (called_allele, NoiseType).

CallSimulator.Simulator — Type

Genotype, config, expression, RNG. Call simulate(sim) for (callsdf, genotypedf, truthphasedf).

CallSimulator.Simulator — Method

Build Simulator from config and params; uses default gene pools. Shortcut for Simulator(config, GenePools(), params).

CallSimulator.Simulator — Method

Build Simulator from config, gene pools, and params (genotype + expression + noise).

CallSimulator.SimulatorConfig — Type
SimulatorConfig(; n_reads, min_unique_vdj=0, seed=0, output_path="calls.tsv", subject_id="sim_donor", noise=NoiseConfig())

Run settings for the simulator. Purpose of main fields:

  • n_reads: how many reads (rows) to generate.
  • min_unique_vdj: if > 0, keep sampling until this many unique (V,D,J) combinations (e.g. for RAbHIT).
  • seed: RNG seed for reproducibility.
  • output_path, subject_id: where to write and how to name the donor in output.
  • noise: call-corruption config (allele swap, D dropout); use NoiseConfigNone for no corruption.
CallSimulator.allele_display_string — Method

Display string for an allele state (show/debug). Present → name, Deleted → em dash.

CallSimulator.alleles_list — Method

Unique allele names present on either chromosome (for genotype table).

CallSimulator.build_calls_df — Method
build_calls_df(subject_id, v_calls, d_calls, j_calls, sequence_ids, d_start, d_end, duplicate_counts; truth columns...)
CallSimulator.build_donor_genotype — Method
build_donor_genotype(donor_id; pool_v, pool_d, pool_j, n_genes_v, n_genes_d, n_genes_j, n_v_het_range, n_j_het_range, p_hemi_*, rng) -> DonorGenotype

Build a donor genotype with typical gene counts and zygosity.

  • Gene counts: Human IGH donors typically have ~38–46 V, ~20–27 D, and 6 J genes. By default the donor gets all genes in the provided pools. Pass n_genes_v, n_genes_d, n_genes_j to use a random subset of that size (e.g. n_genes_v = 42).
  • Zygosity: Heterozygous counts are drawn from n_v_het_range and n_j_het_range. Default n_j_het_range = (1, 1) gives exactly one heterozygous J (typical); use (0, 1) for sometimes 0 sometimes 1, or (1, 2) for 1 or 2. At least one heterozygous J is required when the range minimum is ≥ 1. Hemizygosity is applied with probabilities p_hemi_v, p_hemi_d, p_hemi_j.
  • Co-occurrence: Each simulated read carries exactly one V, one D, and one J; they are always sampled from the same chromosome. So allele co-occurrence is by phase (no cross-chromosome V–D–J pairs).
CallSimulator.build_genotype — Method
build_genotype(donor_id, v_spec, d_spec, j_spec; pool_v, pool_d, pool_j, rng, allele_suffix_hom, allele_suffix_het) -> DonorGenotype

Build a diploid genotype. By default alleles use 01 (homozygous/hemizygous) and *01/02 (heterozygous). Use allele_suffix_het = ("*03", "*04") etc. to get other allele names.

CallSimulator.check_airr_columns — Method

Check whether a DataFrame has core AIRR call columns.

CallSimulator.chr_allele_for_phase — Method

Allele on chromosome for phase table (empty string if deleted).

CallSimulator.compare_with_real — Method

Total variation distance between simulated and real marginals for V, D, J.

CallSimulator.draw_imbalance — Method

Log-uniform draw in [lo, hi] for allele imbalance.

CallSimulator.genotype_table — Method

Genotype table: one row per (donor, locus, gene, allele). Columns: KIcase (donor id), locus, gene, sequenceid (allele name).

CallSimulator.gini — Method
gini(values) -> Float64

Gini coefficient for a distribution of non-negative values (e.g. gene usage counts). Measures inequality: 0 = all values equal (uniform usage), 1 = maximally unequal (one category gets everything). For V/D/J call counts across genes, higher Gini means more skewed usage. Real repertoires often have Gini in roughly 0.3–0.7 for V (many genes, skewed); J can be lower when there are few genes. Simulated data with LogNormalExpr typically give similar ranges.

CallSimulator.ki_donor_preset — Method
ki_donor_preset() -> EmpiricalParams

Tunable parameters with defaults from KI donors: usage skew (lognormalsigma, alleleimbalance), anchor J fraction range, and hemizygosity rates (phemiv/d/j). Example: Simulator(config, GenePools(), ki_donor_preset(allele_imbalance=2.0)).

CallSimulator.maybe_dropout — Method

D dropout: only applies to D locus.

CallSimulator.norm_counts — Method

Normalized count distribution (value -> proportion).

CallSimulator.other_allele — Method

Other allele of the same gene (for allele-swap noise); nothing if no other allele.

CallSimulator.phase_allele_string — Method

String for phase tables: Present → name, Deleted → "".

CallSimulator.random_subset_indices — Method

Indices for a random subset of size n from 1:poollen (or all if n >= poollen).

CallSimulator.sample_locus — Method

Sample one locus for this chromosome: expression → true allele → noise → (call, trueallele, noiselabel).

CallSimulator.summarize_simulation — Method

Summary stats for calibration (nreads, uniquevdj, ddropoutrate, gini per locus, etc.).

CallSimulator.total_variation_distance — Method

Total variation distance between two probability distributions.

CallSimulator.weighted_sample_index — Method

Weighted sample from (index, weight) pairs; returns index.

CallSimulator.zygosity_counts — Method

Count genes by zygosity at locus L. Returns (homozygous=, heterozygous=, hemizygous=).

CallSimulator.zygosity_short — Method

Short label for zygosity (hom / het / hemi).

CallSimulator.NoiseConfigNone — Constant

No corruption (all rates zero). Use with NoiseModel(NoiseConfigNone) when calls should equal ground truth.
