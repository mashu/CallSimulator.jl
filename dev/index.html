CallSimulator

Simulate V/D/J gene call tables (MIAIRR-style) with ground-truth phased genotype for testing phasing and assignment methods. Each read carries one V, one D, and one J call; the simulator records the true chromosome of origin and applied noise.

Design

  • Types (DonorGenotype, GeneEntry{L}, V/D/J): locus-typed genotype; zygosity (homozygous, heterozygous, hemizygous) per gene.
  • GenotypeBuilder (build_genotype, build_genotype_realistic): build genotype from gene pools; realistic donor uses typical counts and zygosity ranges.
  • Expression (ExpressionProfile, build_expression): per-gene, per-chromosome weights; callable to sample a gene for a locus and chromosome.
  • Noise (NoiseConfig, NoiseModel): configurable call corruption (allele swap, gene confusion, D dropout, novel allele). Use NoiseConfigNone for no corruption.
  • Config (SimulatorConfig): read count, seed, paths, noise config, anchor J fraction, duplicate count, cis sigma.
  • Empirical (EmpiricalParams, ki_donor_preset): default gene pools and expression/hemizygosity parameters.
  • Simulator (Simulator, simulate): one chromosome per read; sample V, D, J from that chromosome; apply noise; return calls plus genotype and truth-phase tables.
  • Output (genotype_table, truth_phase_table, build_calls_df, write_simulation_output): MIAIRR-style and ground-truth TSVs.

Output is a MIAIRR-style call table plus reference ground-truth phased genotype; save via the API or CLI.

Why these settings: Expression models skewed gene usage (real repertoires are not uniform); default is LogNormalExpr. Noise models real caller errors (allele swap, gene confusion, D dropout, novel allele) so you can test phasing under tunable error; default is NoiseConfig() with typical rates, or NoiseConfigNone for no corruption. Genotype, expression, and noise are independent: combine any expression method with any noise config (e.g. realistic usage + no noise, or flat usage + default noise). Defaults: gene pools from default_gene_pool(V) or ki_donor_preset(); expression from build_expression(gt; method=LogNormalExpr(1.0), ...); noise from NoiseConfig() or NoiseConfigNone; Simulator(config, ki_donor_preset()) wires genotype and expression from the preset.

Quick start

using CallSimulator
using Random

# Build a donor (typical gene counts)
gt = build_genotype_realistic("donor1"; n_genes_v = 42, n_genes_d = 23, n_genes_j = 6, rng = MersenneTwister(42))
estimated = ki_donor_preset()
cfg = SimulatorConfig(n_reads = 500, output_path = "calls.tsv")
expression = build_expression(gt; rng = MersenneTwister(cfg.seed + 2),
    method = LogNormalExpr(estimated.lognormal_sigma),
    allele_imbalance_range = (1.0 / estimated.allele_imbalance, estimated.allele_imbalance),
    cis_sigma = cfg.cis_sigma)
sim = Simulator(gt, cfg, expression, NoiseModel(cfg.noise), MersenneTwister(cfg.seed))
calls_df, genotype_df, truth_phase_df = simulate(sim)

Or use the preset to build genotype and expression in one go:

config = SimulatorConfig(n_reads = 10_000, output_path = "calls.tsv")
sim = Simulator(config, ki_donor_preset())
calls_df, genotype_df, truth_phase_df = simulate(sim)
